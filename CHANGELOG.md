# Change Log

All notable changes to the "Open With Context Menu" extension will be documented in this file.

## [1.0.0] - 2024-12-XX

### Added
- Initial release of Open With Context Menu extension
- Support for configuring multiple external applications
- File type association with apps (extensions and filenames)
- Cross-platform support (Windows, macOS, Linux)
- Support for both app names and full executable paths
- Context menu integration in Explorer and Editor
- Quick pick menu when multiple apps match a file type
- Automatic app selection when only one app matches
- Configuration via VS Code settings (UI and JSON)
- Configuration change watching for dynamic updates
